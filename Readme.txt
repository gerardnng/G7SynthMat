This repository presents a Grafcet matrix encoding approach for the its synthesis on multy-targets.
This is a Visual Studio Project

It contents :

- The GrafcetConverter tool used for the Grafcet synthesis

- The UniSim tool (http://wpage.unina.it/detommas/unisim/) is integrated to the GrafcetConverter project
  A presentation of UniSim is given with this file : http://wpage.unina.it/detommas/unisim/downloads/AutStrum_Ottobre06.pdf
  
- Some examples of codes generated since some Grafcet models edited with UniSim
